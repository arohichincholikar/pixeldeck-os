# PixelDeck OS

PixelDeck OS is a custom handheld operating system built using Python and Pygame for a Raspberry Pi-powered cyberdeck. Originally developed as a software-first project, it combines reading, creativity, entertainment, and personalization into a single portable device.

The system features a text-based PDF library with reading progress tracking, a persistent music player with playlists and album artwork support, a pixel art studio for creating and saving sprites, a gallery for viewing artwork, and a collection of retro-inspired games including Snake, Tetris, Minesweeper, Flappy Bird, and a Miffy-inspired Tamagotchi virtual pet.

The long-term goal of PixelDeck is to transform this software into a fully portable handheld device powered by a Raspberry Pi, touchscreen display, stylus input, and custom hardware enclosure. The project serves as an exploration of software development, UI design, game development, digital art tools, and embedded hardware integration.

## Features

* PDF Library with reading progress saving
* Music Player with playlists, album covers, shuffle, loop, and resume functionality
* Pixel Art Studio with custom color palettes and sprite export
* Gallery for viewing saved artwork
* Retro Games Collection
  * Snake
  * Tetris
  * Minesweeper
  * Flappy Bird
  * Miffy-inspired Tamagotchi
* Persistent save data and settings
* Custom PixelDeck UI theme and reusable interface components

## Tech Stack

* Python
* Pygame
* PyMuPDF (fitz)
* JSON-based data storage

## Hardware Roadmap

* Raspberry Pi 3-based handheld device
* Touchscreen and stylus support
* Battery-powered operation
* Physical controls
* Custom-designed enclosure
* Optional keyboard integration

PixelDeck OS is an ongoing personal project focused on creating a cozy, all-in-one digital companion for reading, music, drawing, and gaming.
