# PixelDeck OS

A custom handheld operating system built using Python and Pygame.

## Features

- PDF Library
- Music Player
- Pixel Art Studio
- Gallery
- Snake
- Tetris
- Minesweeper
- Flappy Bird
- Miffy-inspired Tamagotchi

## Tech Stack

- Python
- Pygame
- PyMuPDF

## Future Plans

- Raspberry Pi handheld build
- Touchscreen support
- Stylus support
- Custom hardware enclosure
